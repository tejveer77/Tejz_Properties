# Tejz Properties

## Overview
Tejz Properties is a modern web application built with React for managing and showcasing real estate listings. Designed for efficiency and user engagement, it offers live property editing, detailed description modals, and a responsive layout—perfect for real estate professionals or clients browsing properties online.

---

## Features
- **Live Editing**: Update property details (e.g., price, status) in real time without page reloads.
- **Description Modals**: Clickable property cards open modals with full details—clean and intuitive.
- **Responsive Design**: Adapts seamlessly to desktop, tablet, and mobile screens.
- **State Management**: Leverages React hooks for smooth, efficient updates.
- **User-Friendly**: Simple navigation and polished UI for property exploration.

---

## Technologies
- **Frontend**: React, JavaScript (ES6+), HTML5, CSS3.
- **Tools**: Node.js, npm, Vite.
- **Styling**: CSS (Tailwind).

---

## Requirements
- **Node.js**: v16+ (download from [nodejs.org](https://nodejs.org)).
- **npm**: Comes with Node.js.
